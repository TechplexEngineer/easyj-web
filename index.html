EasyJ for FRC Robots

View the Project

EasyJ is based on the Waterbear toolkit. Our goal was to create a tool that would allow beginners the ability to learn the WPI Robotics Library meanwhile having the experience of writing actual code.

Project Tech

  • jQuery
  • HTML 5
  • CSS

Features

  • Draggable Snapping blocks
  • Write nearly syntax free code
  • Explore some fantastic demos
  • Save and restore your projects
