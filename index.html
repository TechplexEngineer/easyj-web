    Originally a 10k Apart entry, Context is a GTD-style task app. It uses HTML5 localStorage to save everything, so you’ll need a modern browser (FF, Chrome, Safari, IE9+). CSS media queries allow the design to adjust to different viewport sizes. While the 10k entry used Typekit for fonts, the version hosted here embeds them manually with @font-face.
